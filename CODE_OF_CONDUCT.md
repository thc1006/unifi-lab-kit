# Code of conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Be respectful. Assume good faith.
- No harassment, discrimination, or personal attacks.
- Keep technical disagreements technical. Argue the patch, not the person.
- If you see unacceptable behaviour, report it privately (see below).
  You will not be retaliated against for a good-faith report.

## Scope

This code of conduct applies in all project spaces: the GitHub
repository (issues, PRs, discussions), any linked chat, and any
offline space where you represent the project.

## Reporting

Use GitHub's private vulnerability-report flow if the issue could
affect other users' security, otherwise open a private issue using
the `confidential` label or email the maintainers via the contact
listed in `SECURITY.md`.

Reports are read by the current maintainers only. We aim to
acknowledge within 3 business days.

## Enforcement

The maintainers may remove comments, close PRs, or ban contributors
for violations. Decisions are made by consensus of the active
maintainers and documented in the relevant GitHub thread.

The full enforcement ladder (warning → temporary ban → permanent ban)
is described in the Contributor Covenant 2.1 document linked above.
